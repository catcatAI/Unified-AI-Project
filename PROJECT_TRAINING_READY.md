## 训练启动步骤

### 1. 运行训练设置脚本
```cmd
setup-training.bat
```

### 2. 激活虚拟环境
```cmd
cd apps\backend
venv\Scripts\activate.bat
```

### 3. 配置训练参数
编辑 `training/configs/training_config.json` 文件

### 4. 启动训练
```bash
python ../training/train_model.py
```
