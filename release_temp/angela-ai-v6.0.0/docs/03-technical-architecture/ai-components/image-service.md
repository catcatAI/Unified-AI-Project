# 圖像生成服务 (Image Generation Service)

## 概述
圖像生成服务 (`image_service.py`) 負責根據文本提示生成圖像。它提供了一個統一的介面，用於圖像生成請求，並抽象化了底層圖像生成模型的細節。

## 當前狀態 (模擬)
目前，圖像生成服务處於**模擬狀態**。它不與任何真實的圖像生成 API 進行互動，而是根據輸入提示返回預設的模擬圖像 URL。這使得 Agent 可以在不依賴外部圖像生成服務的情況下進行開發和測試。

## 核心組件
- **`ImageManager`**: 負責管理圖像生成請求。它提供了一個 `generate_image` 方法，用於根據提示、風格和尺寸生成圖像。

## 整合點
- **`image_service.py`**: 包含 `ImageManager` 的核心邏輯。

## 未來發展
- **真實 API 整合**: 將 `ImageManager` 擴展為整合真實的圖像生成 API (例如 DALL-E, Midjourney, Stable Diffusion 等)，需要配置相應的 API 金鑰和憑證。
- **風格和參數控制**: 實現更豐富的圖像生成風格和參數控制選項。
- **圖像後處理**: 引入圖像編輯、增強和優化功能。
