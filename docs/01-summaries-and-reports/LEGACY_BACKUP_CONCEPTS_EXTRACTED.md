## Security Patterns

### Dangerous Function Detection
- `eval()`, `exec()`, `os.system()` → High severity
- `innerHTML`, `dangerouslySetInnerHTML` without sanitization → High severity
- Hardcoded secrets (api_key, password, token) → High severity
- HTTP (non-HTTPS) API calls → Medium severity

### Recommended Mitigations
- DOMPurify for HTML sanitization
- Environment variables for secrets
- HTTPS-only API calls
